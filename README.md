# Apuntes IA - Android App (Java)

Aplicación móvil diseñada para estudiantes de Informática, orientada a la captura y gestión inteligente de lecciones académicas.

## 📱 Características
- **Grabación Local:** Captura de audio en formato AAC (.m4a) optimizado para voz.
- **Gestión de Materias:** Organización mediante base de datos local (SQLite).
- **Interfaz Moderna:** Diseño basado en Material 3 con soporte para Modo Oscuro.
- **Integración con IA:** Comunicación fluida con un backend de procesamiento para obtener resúmenes automáticos.
- **Reproducción nativa:** MediaPlayer integrado para repasar grabaciones originales.

## 🛠️ Stack Técnico
- Java / Android Studio
- ViewBinding
- OkHttp 3
- SQLite
- Material Design Components
